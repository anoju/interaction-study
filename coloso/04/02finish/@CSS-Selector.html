CSS 셀렉터
* 모든 요소 선택(남발하면 성능에 악영향)
요소이름 해당 요소 모두 선택
p {} p 선택
#foo {} id
#foo id="foo" 선택
.foo {} class
.foo class="foo" 선택
부모  자식 부모에 포함된 자식만 선택
.main-section  .foo class="main-section"에 포함된 class="foo"만 선택
부모 > 자식 한 단계 아래 자식만 선택
.parent > .a 
<div class="parent">
    <div class="a">A</div>
    <div class="b">
        <div class="a">A</div>
    </div>
</div>
a + b a 바로 다음에 나오는 형제 b만 선택
.a + .b 
<div class="b">B</div>
<div class="a">A</div>
<div class="b">B</div>
<div class="b">B</div>
<div class="c">C</div>
<div class="b">B</div>
a ~ b a 다음에 나오는 형제 b를 모두 선택
.a ~ .b 
<div class="b">B</div>
<div class="a">A</div>
<div class="b">B</div>
<div class="b">B</div>
<div class="c">C</div>
<div class="b">B</div>
[속성이름] 해당 속성을 가지고 있는 요소 선택
[class] class 속성이 있는 요소 선택
[속성이름=값] 해당 속성과 값이 일치하는 있는 요소 선택
[class='foo'] class="foo" 선택
[속성이름~=값] 여러개의 값을 가지고 있더라도 속성값에 "값"을 갖고 있다면 선택
[class~='foo'] class="title foo a"
[속성이름*=값] [class*='foo'] class 값에 foo가 포함된 요소 선택
class="foo", class="aafoo", class="bfoo22" 등
[속성이름^=값] [href^='http'] href 값이 http로 시작하는 요소 선택
href="https://studiomeal.com"
[속성이름$=값] [src$='.png'] src 값이 .png로 끝나는 요소 선택
src="aaa.png"
:hover 가상 클래스.
마우스를 요소 위에 올렸을 때
:checked 가상 클래스.
radio, checkbox가 체크된 상태
::before 가상 요소.
.foo::before {
    content: 'before!';
}

<div class="foo">
    before!원래 있는 콘텐츠
</div>
::after 가상 요소.
.foo::after {
    content: 'after!';
}

<div class="foo">
    원래 있는 콘텐츠after!
</div>
